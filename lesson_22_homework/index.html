Задача №1

Що потрапить в консоль?

let someVar = 0;
++someVar;

if (someVar) {
console.log(someVar);
}

Правильна відповідь: 1

- Оператор ++ збільшує значення someVar до 1. Умовне вираження if (someVar) вважає 1 істинним, тому в консоль потрапляє 1.

Задача №2

За допомогою циклу FOR виведіть в консоль 10 рядків:

Пункт №1

Пункт №2

і т.д.

Правильна відповідь:

for (let i = 1; i <= 10; i++) {
console.log(`Пункт №${i}`);
}

- Цикл for проходить від 1 до 10, виводячи рядок для кожного значення.

if (2 * 20 <= 10 || 30 / 2 < 5 && 10 <= "10" || 20 === "20") {
console.log('000');
}

Задача №4

Створіть функцію, яка повертає результат ділення числа a на число b з додаванням рядка "Результат ділення: " Викличте функцію передаючі різні значення, у тому числі не передаючи зовсім. Функція не має повертати NaN, Infinite або помилку

Правильна відповідь:

function divide(a = 0, b = 1) {
return `Результат ділення: ${a / b}`;
}

console.log(divide(10, 2));
console.log(divide(15, 3));
console.log(divide());

- Функція divide приймає два параметри з значеннями за замовчуванням, щоб уникнути NaN або помилки.

Задача №5

Створіть масив даних - 5 елементів, один з яких число 10 Обробіть масив за допомогою методу перебору Перевіряйте елемент на відповідність числу 10, та у разі відповідності, виводьте в консоль

Правильна відповідь:

let dataArray = [5, "hello", 10, 3.14, true];

dataArray.forEach(function(element) {
if (element === 10) {
console.log(element);
}
});

- Метод forEach проходить по кожному елементу масиву і перевіряє, чи дорівнює елемент 10. Якщо так, він виводить його в консоль.

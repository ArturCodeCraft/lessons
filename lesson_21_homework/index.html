Задача №1

Яка назва змінної вірна

1) let user-name

2) let UserName

3) let userName

Правильна відповідь: 3) let userName

- Імена змінних у JavaScript повинні складатися з літер, цифр, символів $, або _. Вони не можуть містити дефіс (-), тому let user-name є некоректним.

- Загальноприйнятим стилем у JavaScript є camelCase для змінних, тобто початкова літера маленька, а наступні слова починаються з великої літери, як у userName. Стиль UserName (PascalCase) зазвичай використовується для назв класів.

Задача №2

Чи буде помилка при виконанні наступної програми

const userName = 20
userName = "Іштван"

Правильна відповідь: Так, буде помилка.

- const оголошує змінну, значення якої не може бути змінене після ініціалізації. Спроба змінити значення змінної userName, оголошеної з const, призведе до помилки TypeError.

Задача №3

Що потрапить в консоль?

let numOne = 20

let numTwo = "20"

let numSumm = numOne + numTwo
console.log(numSumm)

Правильна відповідь: 2020

- У JavaScript, якщо один з операндів операції + є рядком, то інший операнд буде перетворено на рядок, і виконається конкатенація. В даному випадку numOne (число) буде перетворено на рядок і конкатеновано з numTwo (рядок), що дасть результат "2020".

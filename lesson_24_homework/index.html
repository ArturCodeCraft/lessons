Задача №1 (я item, натисни на мене)Я active

Дано в html: три елементи з класом item. При кліку на кожен з елментів додати клас active, при повторному кліку прибрати клас

Правильна відповідь: 1

document.querySelectorAll('.item').forEach(item => {
item.addEventListener('click', function() {
this.classList.toggle('active');
});
});

- Використовуємо метод querySelectorAll для отримання всіх елементів з класом item, і для кожного додаємо обробник події click, який перемикає клас active за допомогою методу classList.toggle.

Задача №4 (я item, натисни на мене) Я active
count:

Дано в html: текст, елемент з класом item, текст. Так, що елемент з класом item за межами в'юпотрта. Створити функцію яка будує інтервал який буде змінювати контент в елементі item виводячи цифру яка збільшується на одиницю: 1 2 3 ... і т.д. Затримка між зміною числа, та до якого числа має працювати інтервал має задаватись в дата атрибутах елемента item. Функція має запустатить коли ми доскролюємо до елементу item (його видно), і не запускатись повторно.

Правильна відповідь:

function isInViewport(element) {
const rect = element.getBoundingClientRect();
return rect.top >= 0 && rect.left >= 0 && rect.bottom <= (window.innerHeight || document.documentElement.clientHeight) && rect.right <= (window.innerWidth || document.documentElement.clientWidth);
}

document.addEventListener('scroll', function() {
const item = document.querySelector('.item');
if (item && isInViewport(item) && !item.classList.contains('counting')) {
item.classList.add('counting');
let count = 0;
const maxCount = parseInt(item.dataset.maxCount, 10);
const interval = parseInt(item.dataset.interval, 10);

const intervalId = setInterval(function() {
if (count >= maxCount) {
clearInterval(intervalId);
return;
}
item.textContent = ++count;
}, interval);
}
});

- Використовуємо функцію isInViewport для перевірки видимості елемента. Якщо елемент видимий і не обробляється, запускаємо інтервал, який збільшує число в елементі з класом item на основі заданих атрибутів data-max-count і data-interval. Інтервал зупиняється, коли досягається максимальне число.

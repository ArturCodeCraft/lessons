Задача №1

Отримати в константу елемент "body"

Правильна відповідь: 1

const bodyElement = document.querySelector('body');

- Використовується метод querySelector для отримання елементу "body" і збереження його в константу bodyElement.

Задача №2

Написати функцію, яка додає в "body" список UL з певною кількістю LI (кількість має передаватись як параметр функції, також мати значення за замовченням)

Правильна відповідь:

function addListToBody(itemCount = 5) {
const ulElement = document.createElement('ul');

for (let i = 1; i <= itemCount; i++) {
const liElement = document.createElement('li');
liElement.textContent = `Item ${i}`;
ulElement.appendChild(liElement);
}

document.body.appendChild(ulElement);
}

// Виклик функції з параметром
addListToBody(10);

// Виклик функції без параметра (буде використано значення за замовчуванням)
addListToBody();

- Функція addListToBody створює "ul" і додає до нього вказану кількість "li" елементів з текстом.

Задача №3

Додати до елементу "body" класс loaded. Потім перевірити чи є клас loaded у елемента "body" і, якщо є, додати стиль кольору тесту зедлений.

Правильна відповідь: Нічого

document.body.classList.add('loaded');

if (document.body.classList.contains('loaded')) {
document.body.style.color = 'green';
}

- Використовується метод classList для додавання класу loaded до "body", потім перевіряється наявність цього класу і змінюється колір тексту.

Задача №4

Дано в html: три елементи з класом item. Треба отримати ці елементи в константу, кожному додати клас active, та перезаписати контент всередені кожного елемента на "Елемент №(тут порядковий номер елементу починаючи з 1)".

Правильна відповідь:

const items = document.querySelectorAll('.item');

items.forEach((item, index) => {
item.classList.add('active');
item.textContent = `Елемент №${index + 1}`;
});

- Метод querySelectorAll отримує всі елементи з класом item, їм додається клас active, і змінюється текстовий контент.

Задача №5

Дано в html: текст, далі кнопка з класом button. Треба прокрутити скрол сторінки до кнопки

Правильна відповідь:

const button = document.querySelector('.button');
button.scrollIntoView({ behavior: 'smooth' });

- Використовується метод scrollIntoView для плавного скролінгу до кнопки.

Задача №6

Дано в html: посилання з класом link Треба додати до посилання дата атрибут зі значенням 100 Поім отримати значення трибуту, та, якщо значення меньше 200 пофарбувати колір тексту посилання в червоний

Правильна відповідь:

const link = document.querySelector('.link');
link.setAttribute('data-value', '100');

const dataValue = parseInt(link.getAttribute('data-value'), 10);

if (dataValue < 200) {
link.style.color = 'red';
}

- Метод setAttribute додає дата-атрибут до посилання, значення отримується і перевіряється, а колір тексту змінюється, якщо значення менше 200.
